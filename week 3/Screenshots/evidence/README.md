# Evidence Collection Guidelines

## Purpose
This directory stores screenshots and evidence collected during assessments.

---

## Categories

### recon
Reconnaissance screenshots

### scans
Scanning and enumeration evidence

### exploitation
Authorized exploitation evidence

### evidence
General proof-of-concept screenshots

---

## Naming Convention

[target]-[activity]-[date].png

Example:
dvwa-nmapscan-2026-05-27.png

---

## Evidence Handling Rules
- Store only authorized testing data
- Avoid exposing credentials
- Maintain timestamp consistency
- Preserve original evidence

---

## Ethical Notice
All evidence collected only in authorized lab environments.
